# 图片处理程序架构

## 流程图

```mermaid
flowchart TD
    A["本地原图目录"] --> B["Rust CLI 扫描图片"]
    B --> C["生成缩略图"]
    B --> D["生成 BlurHash"]
    B --> E["生成 manifest.json"]
    B --> F0["更新增量状态索引"]
    C --> F["输出到本地 .thumbnails/"]
    D --> F
    E --> F
    F0 --> F
    F --> G["外部同步脚本（项目外）"]
    G --> H["Cloudflare R2"]
```

## 技术栈

### 图片处理程序

- 语言：Rust
- CLI：`clap`
- 并行处理：`rayon`
- 图片处理：`image`
- 高性能缩放：`fast_image_resize`
- BlurHash：`blurhash`
- JSON 序列化：`serde` + `serde_json`
- 日志：`tracing` + `tracing-subscriber`
- 增量索引：`state.json`

### 存储与同步

- 对象存储：Cloudflare R2
- 文件同步：外部 `rclone` 脚本
- CDN：Cloudflare CDN
