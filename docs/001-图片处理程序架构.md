# 图片处理程序架构

## 流程图

```mermaid
flowchart TD
    A["source_dir 扫描图片"] --> B["读取 macOS Finder Tag"]
    B --> C{"是否带目标 Tag"}
    C -- "否" --> X["忽略"]
    C -- "是" --> D["解码源图片"]
    D --> E["统一编码为 AVIF"]
    E --> F["写入 originals/"]
    D --> G["生成缩略图"]
    D --> H["生成 BlurHash"]
    F --> I["写入 manifest.json"]
    A --> J["基于 source_dir 更新 state.json"]
    J --> K["清理失效 originals/ 与 .thumbnails/"]
    G --> L["输出到 .thumbnails/"]
    H --> L
    I --> M["构建完成"]
    K --> M
    L --> M
```

## 技术栈

### 图片处理程序

- 语言：Rust
- CLI：`clap`
- 并行处理：`rayon`
- 目录遍历：`walkdir`
- Finder Tag 读取：`xattr` + `plist`
- 通用图像解码：`image`
- HEIF/HIF 解码：`libheif-rs`
- AVIF 编码：`image::codecs::avif::AvifEncoder`
- 高性能缩放：`fast_image_resize`
- 缩略图编码：`webp`
- BlurHash：`blurhash`
- JSON 序列化：`serde` + `serde_json`
- 日志：`tracing` + `tracing-subscriber`
- 增量索引：`state.json`

### 存储与同步

- 对象存储：Cloudflare R2
- 文件同步：外部 `rclone` 脚本
- CDN：Cloudflare CDN
