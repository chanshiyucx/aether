# 数据格式

## 目录结构

以下示例使用当前配置中的 `gallery/` 和 `thumbnails/`，实际目录名以 `aether.toml` 为准。

```txt
sourcePath/
    20240209-Paris/
    20241229-Roma/

targetPath/
    gallery/
    thumbnails/
    manifest.json
    state.json
```

## manifest.json

`manifest.json` 记录对外发布的数据。

- `original.url` 和 `thumbnail.url` 都是相对于 `targetPath` 的路径
- 所有入选图片都会统一转码为 `AVIF` 写入 `gallery/`
- 元数据按“尽力提取”策略写入，缺失字段直接省略

```json
{
  "version": 1,
  "updatedAt": "2026-04-05T12:00:00Z",
  "photos": [
    {
      "original": {
        "url": "gallery/20240209-Paris/IMG_1234.avif",
        "width": 6240,
        "height": 4160,
        "bytes": 1823401,
        "mime": "image/avif"
      },
      "thumbnail": {
        "url": "thumbnails/20240209-Paris/IMG_1234.webp",
        "width": 640,
        "height": 427,
        "bytes": 18234,
        "mime": "image/webp"
      },
      "blurhash": "LrF?lRof00ay~qofWBof_3ayayof",
      "title": "IMG_1234",
      "takenAt": "2025-03-14T18:42:11.123+09:00",
      "location": {
        "lat": 35.6938,
        "lng": 139.7034,
        "alt": 42.1
      },
      "camera": {
        "make": "FUJIFILM",
        "model": "X-T5",
        "lens": "XF35mmF1.4 R",
        "focalLengthMm": 35.0,
        "focalLengthIn35mm": 53,
        "aperture": 1.4,
        "maxAperture": 1.4,
        "shutter": "1/125",
        "iso": 800,
        "exposureProgram": "aperture priority",
        "exposureMode": "manual",
        "meteringMode": "pattern",
        "whiteBalance": "manual",
        "flash": "off",
        "lightSource": "unknown",
        "sceneCaptureType": "standard",
        "brightnessEv": 5.3,
        "sensingMethod": "one-chip color area sensor"
      },
      "image": {
        "orientation": 1,
        "sourceOrientation": 6,
        "colorSpace": "sRGB",
        "hasHdr": true,
        "isLivePhoto": false,
        "bitDepth": 10
      }
    }
  ]
}
```

### 元数据规则

- `takenAt`
  - 优先读取 `DateTimeOriginal`
  - 其次回退到 `DateTimeDigitized`、`DateTime`
  - 如存在 `SubSecTime*` 和 `OffsetTime*`，会一并拼成 ISO 8601 风格时间
- `location`
  - 仅从 EXIF GPS 中提取 `lat`、`lng`、`alt`
  - 不做逆地理编码，因此 `country`、`city` 首版不写入
- `camera`
  - 按实际可提取字段写入，字段级别允许缺省
  - 当前包含一批标准 EXIF 字段：
    `maxAperture`、`exposureProgram`、`exposureMode`、`meteringMode`、`whiteBalance`、`flash`、`lightSource`、`sceneCaptureType`、`brightnessEv`、`sensingMethod`
  - `exposureMode` 当前归一为短值：`auto`、`manual`、`bracket`
  - `whiteBalance` 当前归一为短值：`auto`、`manual`
  - `flash` 当前归一为短值：`off`、`on`、`auto`、`auto-fired`、`red-eye`、`unsupported`
- `image`
  - `orientation` 表示生成后输出图像的方向，当前固定归一为 `1`
  - `sourceOrientation` 在源文件存在 EXIF 旋转信息时可选写入
  - `colorSpace` 只表示色彩空间，不再混入 HDR 语义
  - `bitDepth` 来自源文件探测
  - `hasHdr` 当前按 `bitDepth > 8` 判断
  - `isLivePhoto` 首版固定为 `false`

## state.json

`state.json` 记录增量构建状态。键是 `sourcePath` 下的源文件相对路径，判断基准来自源文件，不是 `gallery/`。

```json
{
  "version": 1,
  "updatedAt": "2026-04-05T12:00:00Z",
  "files": {
    "20240209-Paris/IMG_1234.HIF": {
      "pipelineVersion": 1,
      "size": 8432190,
      "mtimeMs": 1741945331000,
      "original": "gallery/20240209-Paris/IMG_1234.avif",
      "thumbnail": "thumbnails/20240209-Paris/IMG_1234.webp",
      "processedAt": "2026-04-05T11:58:21Z"
    }
  }
}
```
