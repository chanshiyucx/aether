# 数据格式

### 目录结构

```txt
originals/
    20240209-Paris/
    20240213-Palma/
    20241116-Barmsee/
    20241201-Brauneck/
    20241208-Nymphenburg/
    20241229-Roma/
.thumbnails/
manifest.json
state.json
```

## manifest.json

`manifest.json` 记录图片数据，包含原图路径、略缩图路径、blurhash，后续还要包含摄影的元数据，照片元数据（EXIF、GPS、相机参数）、缩略图路径、blurhash 占位、Live Photo 视频引用、HDR 等信息：

```json
{
  "version": 1,
  "updatedAt": "2026-04-04T12:00:00Z",
  "photos": [
    {
      "original": {
        "url": "originals/20240209-Paris/IMG_1234.HIF",
        "width": 6240,
        "height": 4160,
        "bytes": 8432190,
        "mime": "image/heif"
      },
      "thumbnail": {
        "url": ".thumbnails/20240209-Paris/IMG_1234.webp",
        "width": 320,
        "height": 213,
        "bytes": 18234
      },
      "blurhash": "LrF?lRof00ay~qofWBof_3ayayof",
      "title": "IMG_1234",
      "takenAt": "2025-03-14T18:42:11+09:00",
      "location": {
        "lat": 35.6938,

```

## state.json

`state.json` 记录每张已处理图片的指纹：

```json
{
  "version": 1,
  "updatedAt": "2026-04-04T12:00:00Z",
  "files": {
    "originals/20240209-Paris/IMG_1234.HIF": {
      "size": 8432190,
      "mtimeMs": 1741945331000,
      "thumbnail": ".thumbnails/20240209-Paris/IMG_1234.webp",
      "processedAt": "2026-04-04T11:58:21Z"
    }
  }
}
```
