# 数据格式

### 目录结构

```txt
source_dir/
    20240209-Paris/
    20241229-Roma/

path/
    originals/
    .thumbnails/
    manifest.json
    state.json
```

## manifest.json

`manifest.json` 记录对外发布的数据。所有入选图片都会统一转码为 `AVIF` 写入 `originals/`。

  "version": 1,
  "updatedAt": "2026-04-05T12:00:00Z",
  "photos": [
    {
      "original": {
        "url": "originals/20240209-Paris/IMG_1234.avif",
        "width": 6240,
        "height": 4160,

        "mime": "image/avif"
      },
      "thumbnail": {
        "url": ".thumbnails/20240209-Paris/IMG_1234.webp",
        "width": 640,

## state.json

`state.json` 记录增量构建状态。键是 `source_dir` 下的源文件相对路径，判断基准来自源文件，不是 `originals/`。

```json
{
  "version": 1,
  "updatedAt": "2026-04-05T12:00:00Z",
  "files": {
    "20240209-Paris/IMG_1234.HIF": {
      "size": 8432190,
      "mtimeMs": 1741945331000,
      "original": "originals/20240209-Paris/IMG_1234.avif",
      "thumbnail": ".thumbnails/20240209-Paris/IMG_1234.webp",
      "processedAt": "2026-04-05T11:58:21Z"
    }
  }
}
```
